# Tinote Desktop/Web

Lightweight cross-platform planner/note-taking app (Client for MacOS, Windows, Linux)

## prepare to build

```bash
$ npm i -g @neutralinojs/neu
```

## how to build

#### Desktop:
```bash
$ neu build
```

#### Web:
```bash
$ npm run buildW
```

## how to start

#### Desktop:
Just run the one of following executables:
- tinote-desktop-linux
- tinote-desktop-mac
- tinote-desktop-win.exe

#### Web:
```bash
$ npm run startW
```
